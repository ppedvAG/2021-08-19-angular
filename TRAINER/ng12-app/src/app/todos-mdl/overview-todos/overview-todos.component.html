Übung M004

  Aufgabe:
  Erstellen Sie eine Komponente, die die Inhalte von einem Todo-Objekt rendert.

  todo =
  userId: 1,
  id: 1,
  title: "delectus aut autem",
  completed: false

  Bereiten Sie eine Klasse 'done' vor. Wenn bei todo.completed true steht, dann wird die Klasse angewendet.

  CSS-Klasse könnte aussehen
  .done text-decoration: line-through;

Übung M006 Lifecycle

  Legen Sie eine neue Komponente 'date-time' an.

  Die Komponente zeigt Datum und Uhrzeit an.
  Uhrzeit wird jede Sekunde aktualisiert.



  Info:
  Aktualisieren einer Komponente im View, lässt die ganze View neu erstellt werden
